ZWL Calendar

( Code on GitHub )

Usage:

Include the zwl-calendar.js script.

ZWL Calendar exports a global variable (zwlCalendar) with an .init() method. Pass init() the id of a DOM element, and a new ZWL calendar will be inserted into that element. 


<div id="my-calendar \">

<script>
zwlCalendar.init('my-calendar');
</script>
